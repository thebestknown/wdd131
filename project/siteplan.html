Personalized Recipe Hub - Site Plan

Site Name

Personalized Recipe Hub - A website where users can find, save, and customize recipes.

Site Purpose

The site will allow users to explore categorized recipes, search based on ingredients, and save their favorite meals.

Scenarios

  • How can I find recipes based on ingredients I have?
  • Can I save my favorite recipes for later?

Color Scheme

  • Primary Color: Olive Green (#606C38)
  • Secondary Color: Dark Green (#283618)
  • Background: Cream (#FEFAE0)
  • Highlight: Warm Orange (#DDA15E)
  • Accent: Burnt Orange (#BC6C25)

Typography

Primary Font: Playfair Display

Secondary Font: Lora

Wireframe

The wireframe includes a hero section, a search bar, and featured recipes.

Mobile version will have a collapsible menu.
